# The Butterfly Realm

A static website about butterflies built with [Hugo](https://gohugo.io/) and the [Ananke](https://github.com/theNewDynamic/gohugo-theme-ananke) theme.

## Features

- **Home Page**: Introduction to the site.
- **Lifecycle**: Detailed explanation of butterfly metamorphosis.
- **Species**: Profiles of various butterfly species (Blue Morpho, Monarch, etc.).
- **Blue Theme**: Custom CSS overrides for a blue color scheme.

## Development

1.  Install Hugo (Extended edition recommended).
2.  Clone the repository:
    ```bash
    git clone --recurse-submodules <repo-url>
    ```
3.  Run the server:
    ```bash
    hugo server
    ```
4.  Visit `http://localhost:1313`.

## Deployment

The site is configured to deploy to GitHub Pages via GitHub Actions.
